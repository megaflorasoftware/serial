#### Features

- Add release notes (these very ones!)
  - Set up release note pages
  - Add notifier to let users know when new releases go out
- Categories are here, for real!
  - Add and delete categories in the Feeds page
  - Filter videos with the category bar at the bottom of the page
    - _Tip: The bottom categories will be grayed out if no videos match the selected filters. This should make it easy to stay on top of lots of categories!_

#### Improvements

- Moved to a popover for theme editing and color mode selection as opposed to needing a fully separate page
- Change embed URL from **`www.youtube.com`** to **`www.youtube-nocookie.com`** for better user privacy
- Added refresh button to allow for fetching new videos without reloading the window
- Clearer spacing and UI in feeds page
- Data infrastructure simplification
- General project maintenance

#### Bugfixes

- Fixed an issue where adding a feed by channel username would be flaky, resulting in one of the following:
  - An error adding feeds
  - Adding more than one feed

## Features

### Bulk feed active/inactive toggle

- Multiple feeds can now be toggled active or inactive at once from the manage page

### Shift-click range selection

- Hold <kbd>Shift</kbd> and click to bulk select items in the feeds, tags, and views management pages

### Resend email support

- Added [Resend](https://resend.com) as an alternative email provider alongside SendGrid

## Improvements

- Dialogs with long content now scroll properly
- Feed chips in dialogs now paginate when there are many items
- Management page tab style updates
- Support email address is now configurable via `VITE_PUBLIC_SUPPORT_EMAIL_ADDRESS` environment variable

## Other changes

### Progress on implementing subscriptions

- Refined upcoming subscription support in Serial, expanding previous groundwork code changes

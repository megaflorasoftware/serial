## Features

### Keyboard navigation within views

- Use arrow keys to navigate through items in any view
  - In list views, <kbd>↑</kbd> and <kbd>↓</kbd> move through items sequentially
  - In grid views, all four arrow keys work navigating between rows and columns
- Press <kbd>1</kbd>-<kbd>9</kbd> or <kbd>0</kbd> (for the 10th view) to change between active views
- Added new shortcuts for toggling between visibility filters:
  - <kbd>u</kbd> for unread
  - <kbd>i</kbd> for read
  - <kbd>l</kbd> for later

### Keyboard shortcuts for items within views

- Mark items as read/unread with <kbd>e</kbd> or <kbd>Space</kbd>
- Add items to "read later" with <kbd>w</kbd>
- Open items in your preferred open location with <kbd>Enter</kbd>
- Open the original URL in a new tab with <kbd>o</kbd> (in views and when viewing an item)
- Updated the "mark all visible as read" shortcut to <kbd>f</kbd>

## Other changes

- Opening an item now marks it as read. Because of this, recently read items will still be shown in a view until you leave that view
- Grid item heights are now more consistent within a given row

## Improvements

- Desktop actions now only appear on hover, giving more space to page content. You can always show actions by setting "Inline Shortcuts" to "Show" in the Appearance settings, or by holding <kbd>Alt</kbd> to show all keyboard shortcuts.

## Fixes

- <kbd>Alt</kbd> key now only shows shortcuts if it is the only key being held
- Item selection UI no longer shows on mobile, as it is not needed
